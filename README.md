****************************************************************

Overview
This project adds a tabbed conversation system to Rocket.Chat Desktop, allowing users to open and manage multiple chats at the same time.

Problem
Currently, only one conversation can be viewed at a time, which leads to:
Frequent switching between chats
Reduced productivity
Interrupted workflow
Solution

Introduce a multi-tab interface where users can:
Open chats in separate tabs
Switch instantly between conversations
Continue work without losing context
Key Features
Multiple conversation tabs
Session restore (tabs saved after restart)
Keyboard shortcuts
Tab pinning
Smooth and optimized performance

Tech Stack
React
TypeScript
Electron

Goal
To improve productivity and provide a faster, smoother communication experience in Rocket.Chat Desktop.



Conclusion
This feature transforms Rocket.Chat into a multi-tasking communication tool, making it more efficient and user-friendly.
